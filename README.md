### install

1.打開terminal，Clone 此專案至本機電腦
```
git clone https://github.com/Keely1112/ALPHA-SHOP.git
```

2.開啟terminal
```
cd ALPHA-SHOP
```

3.安裝npm套件
```
npm install
```

4.開啟專案
```
npm run dev
```

5.出現以下內容表示伺服器已啟動
```
Local:   http://localhost:5173/
```
